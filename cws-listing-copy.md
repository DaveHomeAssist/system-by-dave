## Full Description

Prompt Lab is a complete prompt engineering workbench that runs entirely in your browser.

Write a prompt, pick an enhancement mode, and get a refined version with variants, quality scoring, and improvement notes — all powered by your own API key. No accounts. No cloud storage. No data leaves your device.

WHAT YOU GET

• Editor with 7 enhancement modes — Balanced, Claude-optimized, ChatGPT-optimized, Image Gen, Code Gen, Concise, and Detailed
• Live prompt quality scoring across 5 dimensions: Role, Task, Format, Constraints, and Context
• Inline word-level diff view showing exactly what changed
• A/B testing — run two prompt variants side-by-side and pick a winner
• Prompt library with search, sort, tag filtering, and collection folders
• Version history — every edit saved, one-click restore to any prior version
• Template variables — use {{variable}} syntax to create reusable prompt templates
• Composer — drag library entries onto a canvas to build multi-block prompts
• Shareable links — export any prompt as a URL, no server required
• Persistent notepad with auto-save, date stamps, and word count
• Command palette (Ctrl+K) and full keyboard shortcut support
• Dark and light themes

PRIVACY-FIRST ARCHITECTURE

Your API key is stored in the extension's isolated service worker — it never touches the page context. All prompt data lives in your browser's local storage. There is no Prompt Lab server. No analytics. No telemetry. No tracking.

BRING YOUR OWN KEY

Prompt Lab works with the Anthropic API. Enter your key in the extension settings and start working. Future updates will add support for OpenAI, Gemini, OpenRouter, and local models via Ollama.

BUILT FOR POWER USERS

Prompt Lab is designed for AI engineers, researchers, writers, and anyone who iterates on prompts daily. It replaces scattered notes, chat history scrolling, and copy-paste workflows with a structured, keyboard-driven workspace that lives in your browser sidebar.
