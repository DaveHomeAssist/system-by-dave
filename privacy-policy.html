Privacy Policy

Prompt Lab · Effective March 30, 2026

The short version: Prompt Lab stores everything locally on your device. Your API keys never leave your browser. We do not collect, transmit, or sell any user data.

What Prompt Lab Does

Prompt Lab is a browser extension for AI prompt engineering. It helps you write, enhance, compare, organize, and version prompts. It connects to AI provider APIs using your own API keys.

Data Storage

All user data — your prompt library, collections, notepad content, preferences, and settings — is stored in your browser's localStorage. This data never leaves your device. There is no Prompt Lab server, database, or cloud sync. If you uninstall the extension or clear your browser data, this data is permanently deleted.

API Keys

Your API key is stored in chrome.storage.local, which is isolated to the extension's background service worker. The key is used exclusively to authenticate requests to AI provider APIs (such as the Anthropic API) on your behalf. Your API key is never transmitted to any server operated by Prompt Lab, never included in analytics or logging, and never accessible to web pages or other extensions.

API Requests

When you use enhancement, A/B testing, or other AI-powered features, Prompt Lab sends your prompt text directly to the AI provider's API endpoint using your API key. These requests go from your browser to the provider (e.g., Anthropic, OpenAI, Google). Prompt Lab does not proxy, intercept, log, or store these requests. The provider's own privacy policy governs how they handle your data.

Data Collection

Prompt Lab collects no analytics, telemetry, usage data, crash reports, or personally identifiable information. There are no tracking pixels, no cookies set by Prompt Lab, and no third-party analytics scripts.

Permissions

Prompt Lab requests only the storage permission, used to securely store your API key in the extension's isolated storage area. No other browser permissions are requested. Prompt Lab does not access your browsing history, tabs, bookmarks, downloads, or any other browser data.

Shareable Links

Prompt Lab can generate shareable links for individual prompts. These links encode the prompt data in the URL fragment (the part after #), which is not sent to any server. Sharing a link shares only the data you explicitly choose to include.

Third-Party Services

Prompt Lab does not integrate with any third-party analytics, advertising, or tracking services. The only third-party communication is between your browser and the AI provider APIs you configure.

Children's Privacy

Prompt Lab is not directed at children under 13. We do not knowingly collect data from children.

Changes to This Policy

If this policy changes, the updated version will be published at this URL with a new effective date. Material changes will be noted in the extension's release notes.

Contact

Questions about this privacy policy can be directed to the Prompt Lab project on GitHub.

Prompt Lab is an open-source project. This privacy policy applies to the Chrome extension, web app, and all distribution surfaces at promptlab.tools.
