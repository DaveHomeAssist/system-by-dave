Policy

Privacy Policy

This policy covers the System by Dave website and its static pages. It does not replace the separate policy for the Prompt Lab product.

Effective March 31, 2026

System by Dave is a static website that showcases Notion skills, agents, widgets, and related tools. The site does not offer user accounts, checkout, or an authenticated application experience.

Short version: this site does not run its own analytics, account system, or database. Most pages are static HTML served through GitHub Pages. When you leave this site for GitHub, NotionWidgets, or Prompt Lab, those destinations operate under their own policies.

What This Site Collects

System by Dave does not directly collect personal information through sign-in forms, checkout flows, comments, or user accounts because those systems do not exist on this site.

Hosting and Basic Web Requests

This site is hosted on GitHub Pages. Like most web hosting platforms, GitHub may process standard request information such as IP address, browser metadata, and request logs as part of delivering the site. That handling is governed by GitHub's own policies, not a custom System by Dave backend.

Third-Party Resources

Some parts of the site load resources from third parties in order to render properly.

  • Google Fonts is used to load the site's typefaces.
  • The widgets page embeds live widget demos hosted on GitHub Pages under the NotionWidgets project.
  • Outbound links may send you to GitHub, Prompt Lab, or other external properties.

Those services may have their own logging, cookies, or privacy practices. You should review their policies if you interact with them directly.

Cookies and Analytics

System by Dave does not currently run first-party analytics, advertising trackers, marketing pixels, or cookie consent tooling because the site does not set up its own tracking stack.

Prompt Lab

Prompt Lab is a separate product with its own privacy policy. If you are using Prompt Lab, refer to promptlab.tools/privacy for the product-specific terms that apply there.

Changes to This Policy

If this page changes materially, the updated version will be published at this URL with a revised effective date.

Contact

Questions about the site can be directed through the project repository at github.com/DaveHomeAssist/system-by-dave.
