02 — Mark anatomy
Three horizontal bars

Equal weight, equal spacing. The rhythm of a structured system — top, mid, bottom.

Alternating vertical connectors

Left side connects top to mid. Right side connects mid to bottom. Traces an S without curves — pure grid logic.

20px grid unit

Every element is a multiple of 20px. Bars are 20px tall. Connectors are 20px wide. Nothing arbitrary.

Open corners

Top-right and bottom-left stay open. This is what makes it an S and not a grid block.

07 — Brand rules
DO
  • Use "System by Dave" in full on first reference
  • Use the geometric S mark at 40px minimum size
  • Keep the rust accent as the single color accent
  • Use Syne 800 for all display headings
  • Pair DM Mono labels with DM Sans body text
  • Use the warm dark bg (#1c1917) not pure black
  • Maintain the grid aesthetic across all assets
  • Use parchment (#f2ede6) for light mode backgrounds
DON'T
  • Use lime green or any neon accent color
  • Stretch or distort the S mark proportions
  • Use more than one accent color at once
  • Set the wordmark in any font other than Syne
  • Use the mark below 24px — use text only
  • Place the logo on busy photographic backgrounds
  • Use rounded corners on the mark or grid elements
  • Mix the rust and lime green palettes
